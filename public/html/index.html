Use this application to create a customer order.
Once submitted the form will create entries into both the Customer Order and Customer Order Items tables.
A customer can purchase up to 10 items per order.
Use this application to create new Customers and view existing Customer data.
The "Add New" function will Insert an entry into the Customers table.
Use this application to create new Customer Orders, view existing Customer Orders, Edit an existing Customer Order, or Delete an existing Customer Order.
Use this application to insert Items on a Customer Order, view existing Customer Order Items, and Search by Order ID.
Typically, the Items on a Customer Order will be entered in the New Customer Order application, and this can be used to add additional Products to an existing Order.
Use this application to create new Employees and view existing Employee data.
Use this application to create new Products, view existing Product data, or Edit an existing Product.
